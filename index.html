Slinky Toolbox

Build delightful web frontends with this powerful and configurable toolbox
Start Creating

Configurable

Toolbox is fully customizable and extensible. Customize CSS and Javascript linting, spritesheet generator, image optimization, input and output file system, install aditional PostCSS and Gulp plugins.

CSS / PostCSS

Write future CSS syntax with PostCSS using postcss-preset-env plugin, optimized with CSSNano and linted with Stylelint. Other plugins like Lost Grid and postcss-mixins are also included.

Javascript

Write Javascript with consistent code style that is also optimized for production using true minification and optimization. Javascript is also linted usingJSRC and JSHint.

Assets Workflow

Toolbox also includes image minification and optimization, automatic svg spritesheet generation, automatic favicon generation and font handler. Slinky Toolbox has got you covered.

1. Get The Toolbox

Slinky Toolbox on GitHub

2. Tune it to your liking

./gulp.config.js

3. Run it and start creating

npm run dev:watch
